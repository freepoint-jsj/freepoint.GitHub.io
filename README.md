# 关于建站的想法 | About Website
写作的初衷并不是奇淫技巧的炫耀，也不是单纯为了流量赚钱。有句话叫：` “把问题写出来问题就解决了一半！” `,从这个角度来说，有些东西能看懂并一定能写出来，能写出来不一定让读者立刻就明白，只有通过自己的思考加输出，日积月累的可以锻炼，才能不断进步。
